## Installation

Download with your favorite package manager.  

With Packer:
```lua
use "lewpoly/sherbet.nvim"
```
## Requirements

- neovim >= 0.6
- truecolor terminal support
- undercurl terminal support (optional)

## Usage

As simple as writing (pasting)

```vim
colorscheme sherbet
```

```lua
vim.cmd("colorscheme sherbet")
```
